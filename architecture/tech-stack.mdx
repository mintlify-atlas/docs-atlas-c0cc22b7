## Styling

### Tailwind CSS 4.1.13

Next-generation Tailwind with Oxide engine.

<Accordion title="Tailwind v4 Features">
**New in v4:**
- **Oxide Engine:** 10x faster compilation
- **Zero Config:** Works out of the box
- **Native CSS:** No more purge configuration
- **PostCSS Plugin:** `@tailwindcss/postcss@^4.1.13`
- **Better IntelliSense:** Improved autocomplete

**PostCSS Configuration:**
```javascript
// postcss.config.mjs
export default {
  plugins: {
    '@tailwindcss/postcss': {},
  },
}
```
</Accordion>

<Accordion title="Tailwind Plugins">
- `@tailwindcss/typography@^0.5.16` - Beautiful prose styling
- `@tailwindcss/forms@latest` - Form element styling
- `tailwind-merge@^3.3.1` - Merge conflicting classes
- `tailwind-variants@^3.0.0` - Component variants
- `tw-animate-css@^1.3.8` - Animation utilities
- `tailwind-scrollbar-hide@^4.0.0` - Hide scrollbars
</Accordion>

<Accordion title="Design System">
**Utility-First Approach:**
```tsx
<button className="rounded-lg bg-blue-600 px-4 py-2 text-white hover:bg-blue-700">
  Click me
</button>
```

**With Class Variance Authority:**
```tsx
import { cva } from 'class-variance-authority'

const button = cva('rounded-lg px-4 py-2', {
  variants: {
    intent: {
      primary: 'bg-blue-600 text-white hover:bg-blue-700',
      secondary: 'bg-gray-200 text-gray-900 hover:bg-gray-300'
    }
  }
})
```
</Accordion>

### Zustand

<Accordion title="Example Store">
```typescript
import { create } from 'zustand'

interface Store {
  user: User | null
  setUser: (user: User) => void
}

const useStore = create<Store>((set) => ({
  user: null,
  setUser: (user) => set({ user })
}))
```

**Benefits:**
- Minimal boilerplate
- TypeScript support
- No providers needed
- Middleware support
</Accordion>

### GraphQL Code Generator 6.1.2

Type-safe GraphQL client generation.

<Accordion title="Stack">
- `graphql@16.13.0` - GraphQL implementation
- `graphql-request@7.4.0` - Lightweight GraphQL client
- `@graphql-codegen/cli@6.1.2` - Code generator
- `@graphql-codegen/typescript@5.0.8` - TypeScript types
- `@graphql-codegen/typescript-operations@5.0.8` - Query types
- `@graphql-codegen/typescript-react-query@^6.1.0` - React Query hooks
</Accordion>

<Accordion title="Workflow">
1. Write `.graphql` query files
2. Run `bun run codegen`
3. Import generated hooks

```typescript
// Auto-generated, fully typed
import { useGetOrganizationQuery } from '@repo/codegen/src/schema'
```
</Accordion>
